# Backend

Express API for the Golf Charity Subscription Platform.

## Scripts

- `npm run dev` - start in development with nodemon
- `npm start` - start in production mode

## Main Modules

- `auth` - register/login/me
- `user` - subscription, scores, dashboard
- `draw` - simulation, publish, results
- `admin` - analytics, users, winners, charities
- `charities` - listing and filtering
